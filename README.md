# Tartuntatiedot

Dashboard application that is meant to track the COVID-19 situation around the globe and particularly in Finland.

http://users.metropolia.fi/~kerttulr/tartuntatiedot.fi/

### Tech Stack

- *JS* 
- *REST API*
- *HTML5*
- *CSS3*
